How to create a personal page with GitHub

This is a step-by-step tutorial of how to set up a personal web page. We will start off with a very basic structure, but will include features that you can add to later down the track. You will need the following:

    Computer Internet Access Internet Browser (i.e. google chrome) Code editing software. (We will be using “Visual Studio Code”) A GitHub account

The tutorial will be broken into three parts. Creating the folders and files with structure necessary for web development; Writing html code using Visual Studio Code; and uploading your files upload to a server, in this case “GitHub”.

First, we will look at setting up the files needed for a website.

>

(Note: For an in-depth look at what makes up a website, follow the link to MDN:Dealing with files)

Start by creating a directory somewhere easy to find, (i.e. desktop), and name it something like “web-projects”. (Note: a “directory” is the coding term for folder on a computer).

We will use this directory to store everything that will be on your website. Make three directories within the new directory: images; styles; script. (Note: avoid spaces in file names. A good practice is to use “–“ instead of a space, i.e. “my dog”=”my-dog”).

Place any images that you want on your webpage (i.e. profile picture) in the folder named…….you guessed it, images.

Now that our files are complete, we are ready to upload to an online server.

Open GitHub and follow the instructions to create your online profile.

Now that you have a profile you will need to create a repository. (This is what you will type into the browser)

    1. Click on the round icon on the top right of your window containing your avatar (this will be patterns if you are just logging in for the first time).

    2. This will open a dropdown list, click “Your repositories”.

    3. Click on the green “New” button.

    4. 4. Enter a repository name, then “Create Repository”.

    5. Click on “upload an existing file.”

    Open up your file browser, then drag and drop the “web-projects” directory that you made earlier, into the box.

You can now test your personal webpage by typing your repository from earlier into the browser.

Congradulations. You have the building blocks to your personal webpage.
